YD周一：你知道多少种XXOO类型？

我(原文作者)一直在琢磨一个问题：XXOO到底有多少种类型？没什么不好意思的。

废话不多说，我们开始吧。

初次炮
一般情况下，第一次大家都笨手笨脚，不能尽兴，要是你不懂得如何呻吟，多半第一次不会让你们满意。完事之后，正确的安慰语应该这么说：高潮不是你想要就有的。

管你是谁炮
光看名字，基本上能猜个八九不离十。这种炮只是为了好玩，不过，和一个你完全不认识的人来一炮，结果很可能是一点都不好玩。实际上这种约炮方式错的太离谱了，我该从哪儿开始吐槽呢？虽然我从来没体验过疯狂愚蠢大汗淋漓的一夜情，听上去好像我很可怜似的，不过我也是有过几次机会的，但是在这种情况下，我总是先认真的考虑这样做的后果，考虑一次还不放心，还要再考虑一次……我知道，我就跟我妈一样！

热恋初期炮
当两个开始了解彼此，关系到了一定程度，深知彼此的需求，就可以用这种方法愉悦对方。女人们穿着性感内衣，刮干净脚上的腿毛，男人们把大肚腩用力收起，喷上昂贵的古龙水。听上去好像大家都有所斩获。完事后，她的疯婆子本性暴露无遗，他的“金枪不倒”不攻自破。很多人把第一次当成真爱炮，不过两者连边都搭不上。

真爱炮
真爱炮的可贵，在于高潮和高潮之后的抱抱。要找到真爱你需要花一番功夫，真爱炮会让一切辛苦变得值得。两个人不但让彼此获得性的愉悦，还能在接下来的23小时56分钟里，变着法儿地讨爱人欢心。

懒炮
大家都经历过懒炮吧。很久没有惊天动地的干一番，感觉感情基础快要完蛋，赶快得做些什么弥补一下。但是你们两个谁都不想动，不想出汗，不想搞得自己太累，怎么办？“你躺在床上，我也刚好躺在床上，那就开始吧。”有过一次懒炮，恋情就基本完蛋了。

偷欢炮
偷欢既刺激又热辣，还让人欲罢不能。许多婚姻都是因为偷欢破裂的，幸运的是到目前为止我对我的丈夫还算忠诚。但是，只要想一想偷欢要付出的代价——丧失真爱和家庭，怎么想怎么不划算。我看到过许多家庭因为不忠闹得鸡飞狗跳，没有一个好下场。

和好炮
吵架中的两方想和好，想做些补偿，想弥合爱情的裂缝。所以，一般情况下，和好炮的双方都是诚心想要的，不但如此，激烈程度猛增，还能做到十分贴心。完事后，是亲亲秘密的呢喃，约好以后再不吵架。直到下一次，他抱怨她的衣柜塞得太乱，她抱怨他又在喝酒。不过，想想和好炮吧，吵架是值得的。

怜悯炮
这种炮可以算尽义务，是，这种算不上最好的，不过你会因此带上“他/她人还不错”的帽子，是不？

孩子不在家炮
孩子们出门了，父母们可以放心干。狂野、吵闹、从厨房到厕所，从客厅到卧室……其实这应该叫“好久没来，终于让我们等到了”炮。

打灰机
这种是最简单的了，虽然能获得满足，但是没有人可以谈心，可以呻吟，可以紧紧抱住，这些都是打灰机的缺点。但是总比管你是谁炮好得多。

三人行
我问了所有人，有超过40%的人表示没有玩过三人行，我自己也没玩过，只有靠你们自己想象了。

S&M
无非是橡胶制品、绑人的锁链、手铐和挠痒痒的羽毛……
